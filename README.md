# EnhancedEditTexts
A custom EditText set

# 使用
### Step1 Add it in your root build.gradle at the end of repositories:
      ```allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}```
### Step2 Add the dependency
      ```dependencies {
	        compile 'com.github.ysutommy:EnhancedEditTexts:-SNAPSHOT'
	}```
